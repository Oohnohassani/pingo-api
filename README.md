
# Pingo API 🌍

A simple mock API for the Pingo travel app, powered by json-server.

### 🚀 Getting Started

### 1. Install dependencies
```bash
npm install
````

### 2. Run the server

```bash
npm start
```

### 📡 API Details

* Base URL: [http://localhost:10000](http://localhost:10000)
* Port: 10000
* Host: 0.0.0.0

The API serves data from:

```
data/places.json
```

### 📁 Example Endpoints

* `/places` – Get all places
* `/places/:id` – Get a single place

### 🛠 Tech Stack

* json-server

### 📌 Notes

This is a lightweight mock API intended for development and testing purposes.
